# Handoff: 学習塾フォーカス（FOCUS）公式サイト リニューアル

## Overview
勉強が苦手な中学生専門の個別指導塾「フォーカス」の集客サイト。広告・検索から来た保護者（主に母親、スマホ利用9割）を、資料請求（SMS配布）・無料相談・電話につなげることが目的。Astro静的サイト（リポジトリ: `focus/website`）での実装を想定。

## About the Design Files
このバンドル内のファイルは **HTMLで作られたデザインリファレンス**（見た目と挙動を示すプロトタイプ）であり、そのまま流用する本番コードではない。タスクはこれらのデザインを **対象コードベース（Astro）の既存の構成・慣習で再実装する** こと。`.dc.html` ファイルはカスタムランタイム（`support.js`）上で動くコンポーネント形式のため、テンプレート部分（`<x-dc>`内のHTML＋インラインstyle）とロジッククラス（状態・ハンドラ）を読み取り、Astro/バニラJS（またはプロジェクトで選定したフレームワーク）に移植する。`{{ }}` はテンプレートホール、`sc-if`/`sc-for` は条件分岐/繰り返し、`style-hover` 等は擬似クラスに読み替える。

## Fidelity
- **トップページ（トップページ.dc.html）: High-fidelity**。色・文字・余白・インタラクションは最終仕様。ピクセル忠実に再現する
- **相性診断（相性診断.dc.html）: High-fidelity（ロジックは仮実装）**。配点はREADME記載のものが正
- **ワイヤーフレーム.dc.html: Low-fidelity**。下層ページ（/reason /method /price /faq /classrooms/... /document /consult /company）の構造ガイド。スタイリングはトップページのトークンを適用する
- 診断ポップアップ案.dc.html／特徴2案.dc.html／料金比較案.dc.html 等は検討過程の資料（採用案はトップページに反映済み）
