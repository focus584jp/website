# FOCUS 公式ホームページ 設計仕様（フレームワーク先行）

作成日: 2026-06-26
リポジトリ: `focus/website`

---

## 0. この仕様のスコープ

本仕様は **FOCUS（個別指導塾）の公式HP全面刷新** の設計を定める。
進め方は「**フレームワーク（構造）を先に固め、後からデザインを当てはめる**」フェーズ分離方式。

- **Phase 1（本仕様で固める）= フレームワーク**: ページとセクションの並び・各セクションの役割・CV（コンバージョン）位置を確定。中身は見出しレベルのダミーで可。Astroの技術骨組みもここで用意する。
- **Phase 2 = コピー**: 各セクションの実文章を初回面談トークスクリプトから起こす。
- **Phase 3 = デザイン**: design-systemを吟味した上で視覚デザインを当てはめる。

※ **design-systemの吟味は別トラック**として進める。本仕様ではデザインを確定させない（方向性のみ記録）。

---

## 1. 目的・背景

- 古いHPを全面刷新する。Google/Yahoo広告・自然検索からの流入の**着地点**として機能させる。
- ゴール: 「フォーカスってどんな塾だろう」と調べた保護者が、**「とりあえず一度話を聞きに行ってみよう」**と思える内容にする。
- 主CV（問い合わせまで）: **資料請求（デジタルパンフをメール即送付）／無料相談（来塾1時間）／電話**。

## 2. ターゲット・ペルソナ

主役は **保護者＝母親**。問い合わせまでは母親に集中、生徒本人の「ここなら通えそう」は補助的に添える。

**メインペルソナ: 母親（40代前半）**
- 子は公立中の中2前後、定期テスト平均〜やや下（偏差値45〜55帯）、数英が苦手。
- 不安: ①月謝が高いと続かない ②安い塾は講師の質が心配 ③うちの子みたいな"できない子"が浮かないか ④本人が嫌がって続かない。
- 求める一歩: いきなり申込はこわい。**まず料金を知りたい／一度話を聞いて雰囲気を確かめたい**。

**サブ視点: 生徒本人（勉強が苦手な中学生）**
- 「怒られたくない」「分からないと言いづらい」。HPでは優しい雰囲気・講師との距離の近さを伝え、母親が「これなら本人も嫌がらないかも」と感じる材料にする。

## 3. ポジショニング

**一言で**: 「勉強が苦手な中学生のための塾。千葉大生がつきっきりで、大手個別の"約半額"。」

**カテゴリ**: 千葉市の「勉強が苦手な中学生（偏差値45〜55）専門」の個別指導塾（最大1対4）。進学エリート塾とは戦わない。

**3本柱（優先順位）**:
1. **苦手な子"専門"** — 「分からないが当たり前」から始める。対象を絞るから寄り添える（共感で自分ごと化）。
2. **通いやすい価格・明朗会計** — 1教科5,980円（税抜）。5教科でも集団塾並み（3〜3.5万）。入会金・管理費なし、教材費＋システム料 月1,250円のみ。
3. **講師の質（千葉大生）** — 全員千葉大生。低価格塾の"質の不安"を打ち消す信頼装置。

**補強**: 千葉県公立入試（内申・定期テスト対策）に直結／千葉市10校舎の地域密着。

## 4. 市場・競合インサイト（調査要約）

- **相場**: 中学生の個別指導は1教科 月14,000〜16,500円＋諸経費が一般的（明光・スクールIE等）。FOCUSの5,980円は**大手1対2（森塾 月11,700円〜）の約半額**水準。
- **価格の見せ方**: 明光・スクールIEは料金非公開。森塾のみ「5,880円〜（税込）」を明示（FOCUSの税抜5,980円と見出し額が近接）。→ **単純な安さではなく「諸経費込みの総額」「5教科でも集団塾並み」で語る**のが誠実かつ強い。
- **差別化の構図**: 料金非公開・総合型（明光/スクールIE）に対し **料金明示＋ターゲット特化**／万人向け1対2・成績保証（森塾）に対し **苦手な子に寄り添う安心感**／プロ講師訴求（エソー等）に対し **身近な地元の千葉大生**。
- **千葉市の受験事情**: 公立志向＝**内申・定期テスト対策**が刺さる。
- **面談トークスクリプトの説得順**（HP構成の土台）:
  1. 塾の選び方（相性診断）= 通信→集団→個別1対2→「だから1対4」と消去法で誘導（**競合HPにない独自コンテンツ**）
  2. 苦手な子専門の宣言（平均点突破からやり直す）
  3. 価格の決定打（5教科でも集団塾並み／明朗会計）
  4. 指導方針（解き方でなく"勉強のやり方"を教える＝自立。1対4だからこそ）
  5. 柔軟な受け方（苦手科目を週2回、テスト前ピンポイント、部活両立）
  6. 体験授業でノーリスク確認 → CV

## 6. Phase 1 で固める各ページのワイヤー（セクション並び）

中身は見出しレベルのダミーで可。**並び・役割・CV位置**を確定するのが目的。

**トップ（広告着地点。長尺1ページで全体像）**
1. ヘッダー（ロゴ／ナビ／教室を探すアイコン／資料請求・無料相談ボタン）
2. ヒーロー（苦手専門の宣言＋母親への添え書き＋資料請求/無料相談CTA）
3. ページ内アンカーナビ（選ばれる理由・相性診断・料金・実績・教室へジャンプ）※伊藤塾LPの学び
4. こんな悩みありませんか？（チェックリストで自分ごと化）※北斗LPの学び
5. 選ばれる理由 3本柱（苦手専門→価格→千葉大生）
6. 塾の選び方（相性診断）ダイジェスト → 詳細ページへ
7. 料金ダイジェスト（5,980円を大きく／5教科でも集団塾並み）
8. 指導方針ダイジェスト（勉強のやり方を教える＝自立）
9. 合格実績・生徒/保護者の声
10. 教室マップ＋一覧 → 各教室へ
11. 入塾までの流れ（01→02→…）
12. FAQ（抜粋）
13. CTAバンド（資料請求・無料相談・電話）
14. フッター（会社概要・プライバシーポリシー等）
15. 追従CTA（スマホ下部固定＋PC常設）

**下層ページ**は各「目的」に沿ってセクションを構成（詳細はPhase 1実装時に確定）。
- 相性診断: 通信／集団／個別1対2／1対4の比較・診断形式＋CV
- 料金: 通常授業料／講習／諸経費（明朗会計）／柔軟な受け方／他塾比較／CV
- 指導方針: Teaching vs Coaching図解／基礎から積み上げ／自立／CV
- 教室ページ（テンプレート）: 後述データモデル参照

## 7. CV導線設計

優先度（ペルソナの"軽い一歩"重視）:
1. **資料請求**（デジタルパンフをメール即送付）＝最も低ハードル → オレンジCTA
2. **無料相談**（来塾1時間）→ ネイビーCTA
3. **電話**（各教室直通）→ グレーCTA

- 全ページに**追従CTA**（PCヘッダー常設＋スマホ下部固定バー）。参考サイトで最も支持された要素。
- CTAは各セクション末にも反復配置（LP定石）。
- **LINE相談は保留**: FOCUSのLINE公式（相談用）へメッセージを送る導線。採否は後で判断するため、構造上は「差し込める余地」だけ残し、Phase 1では主CVに含めない。

## 8. 教室ページのデータモデル（ローカルSEOの肝）

10校舎は**共通テンプレート＋教室データ流し込み**で生成（手書き10ページは保守不能のため不可）。

教室データ1件の項目（案）:
- 教室名 / スラッグ（URL用）
- 住所 / 最寄駅・徒歩分数 / Googleマップ埋め込み or 座標
- **対応中学校名リスト**（地域検索流入の要）
- 教室責任者（氏名・顔写真・ひとこと）
- 教室写真（複数）
- 直通電話番号
- 開校時間・対応学年
- 空き状況・補足（任意）

→ Astroの**コンテンツコレクション**（または `data/classrooms.*` の配列）から `[slug]` 動的ルートで10ページ生成。

## 9. デザイン方針（Phase 3で適用・ここでは確定しない）

**結論**: FOCUSのdesign-systemは既に現トレンドと方向が合致。**design-systemを土台に「最新っぽさ」を足し込む**。主軸リファレンス＝**コノ塾（すっきり見やすい）**。安心感×今っぽさのバランスは**RED＋シロヤギ**のいいとこ取り。

参考8サイトから採用したい要素（Phase 3で具体化）:
- 追従CTA（スマホ下部固定＋PC常設）— 四街道/キャンパス/RED
- 下からのフェードイン（控えめ）— atama+FC/RED
- 大きな数字での料金・実績表示（`.bignum`）— RED/コノ塾
- 斜め構図の大型ヒーロー＋写真＋手書き添え書き — 北斗/シロヤギ/コノ塾
- トップのページ内アンカーナビ — 伊藤塾
- 「こんな悩みありませんか？」チェックリスト — 北斗
- 方眼グリッド（`.grid-bg`、塾らしさ）— シロヤギ
- 人物イラスト — atama+FC
- ヘッダーに教室検索アイコン — RED

**あえてやらない**: 過剰アニメ／グラスモーフィズム／グラデーション（design-systemで廃止済み）。安心感を壊さないため動きは控えめ、余白と階層で見せる。

**デザイン制作の進め方**: Phase 3では Claude Code で実装し、**ヘッドレスChromeのスクリーンショットで見た目を確認しながら反復**。ヒーロー等は複数案を作って見比べる。写真・人物イラストはプレースホルダーで作り、実素材は別途差し替え。

## 10. 技術構成

- **フレームワーク**: Astro（静的サイトジェネレータ）。コンポーネント分割・`astro:assets`によるWebP最適化・モバイルファースト・セマンティックHTML。
- **スタイル**: design-system（`focus-ui.css` ＝ `tokens.css` を内包）を読み込む。コピーせず参照／同期方針はdesign-systemルールに従う。
- **公開**: GitHub Pages（無料。欠席フォームと同じ運用の延長）。
- **データ**: 教室情報はコンテンツコレクション／データファイルで一元管理。
- **コスト**: 制作・公開は¥0。独自ドメイン（年数千円）は任意で公開前に判断。将来のWordPress移行時もマークアップ資産が下敷きになる。

## 11. スコープ外・未決事項

- **独自ドメイン**: 公開前に別途相談（最初は `*.github.io` でも可）。
- **LINE相談**: 採否保留（FOCUS LINE公式へ送る導線）。
- **価格表記の統一**: 外部ポータルに旧価格「9,980円」が残存。新価格5,980円との表記統一は別タスク（HP側は5,980円で統一）。
- **写真・人物イラストの調達**: 自前撮影／発注／AI生成は別途。Phase 1〜2はプレースホルダー。
- **WordPress移行**: 将来検討。今回は対象外。
- **design-systemの吟味**: 別トラックで実施。Phase 3のデザイン確定前に反映。
- **2026年度千葉県公立入試の変更点**: コンテンツ掲載時に教育委員会の最新要項で要確認。

## 12. 成功基準

- Phase 1完了時: 全ページのセクション構成・CV位置が確定し、Astroで全ページ（教室10ページ含む）がダミー内容で生成・表示できる。
- 最終: 広告/検索流入の保護者が、迷わず資料請求・無料相談に至れる導線と内容になっている。
